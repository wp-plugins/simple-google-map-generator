== Description ==
This plugin provide you a Google map shortcode to use everywhere of your website.
You will select your address just with a simple Drag and drop.
Press BUILD, and you will get the code


Language Support:

* English
* Persian

Features:

* Drag and Drop address selection
* Support Pop up message
* Custom map Icon
* Icon animation
* RTL support


== Installation ==

1. Upload plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Settings' and build your code
